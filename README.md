# 🚀 AI Resume Analyzer Pro

An intelligent Resume Analysis Web Application built using Python and Streamlit.

## 📌 Project Overview

AI Resume Analyzer Pro extracts and analyzes information from resumes (PDF format) and compares it with a given job description.

It provides:
- Skill detection
- Skill frequency analysis
- Job match percentage
- Missing skills identification
- Experience detection
- Education extraction
- Resume strength scoring
- Intelligent improvement recommendations

---

## 🧠 Features

✅ Skill Detection
Extracts technical skills category-wise.

📊 Skill Frequency Analysis
Counts how many times each skill appears.

🎯 Job Match Percentage
Calculates resume compatibility with job description.

❌ Missing Skills Detection
Identifies required skills not present in resume.

💼 Experience Detection
Extracts total years of experience using regex-based NLP.

🎓 Education Extraction
Detects degree information (BS/MS/PhD etc.)

🚀 Resume Improvement Suggestions
Provides automatic improvement recommendations.

---

## 🛠 Tech Stack

- Python
- Streamlit
- spaCy (NLP)
- pdfminer.six
- Regex (Rule-Based Parsing)
